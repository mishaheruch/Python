## Теорія

### Мета: навчитися працювати з текстовими даними.

### 1. Спробуй вивчити функції для роботи з текстовими даними [ось тут](https://docs.python.org/uk/3.13/library/stdtypes.html#string-methods).

## Завдання

У цій папці створи файл `text_analyzer.py`. Цей файл має приймати фрагмент тексту від користувача і аналізувати його. Потрібно визначити:

- кількість слів у тексті
- кількість речень у тексті
- найчастіше повторюване слово
  і вивести це користувачу.

Для того, щоб спростити завдання, будемо вважати, що користувач завжди передає добрий текст.

Наприклад, для такого тексту:  
`Lorem ipsum dolor sit amet. Ullus investigandi veri, nisi inveneris, et quaerendi defatigatio turpis est, cum esset accusata et. Ullus investigandi veri, nisi inveneris, et quaerendi defatigatio turpis est, cum esset accusata.`

має бути така статистика:

`слів: 32, речень: 3, найчастіше повторюване слово: et, 3 рази.`

_Підказка. Тобі допоможе вбудований клас `Counter`. Його можна додати на початку файла ось так: `from collections import Counter`. Якщо передати у Counter текст, то він порахує кількість повторень кожного слова. Почитай документацію щодо можливостей Counter, це допоможе тобі виконати завдання._
