# Неформальне знайомство з Python

## Теорія

### Мета: познайомитися з деякими базовими типами даних в Python і операціями з ними

### 1. Числові типи даних: [дивися ось тут пункт 3.1.1](https://docs.python.org/uk/3.13/tutorial/introduction.html#using-python-as-a-calculator).

### 2. Текстовий тип даних: [розбери п.3.1.2 звідси](https://docs.python.org/uk/3.13/tutorial/introduction.html#text).

Спробуй запустити інтерпретатор python і виконати ті команди, які є в документації. Це допоможе тобі краще зрозуміти, як працює ця мова. Щоб запустити інтерпретатор, відкрий warp, зроби собі нову вкладку і просто запусти `python3`.

## Завдання

У цій папці створи файл `age.py`. Напиши у ньому програму, яка запитує у користувача ім'я і вік людини, та виводить такий текст:

`"Привіт, <тут має бути ім'я>! Через рік тобі буде <вік + 1> років".`

Наприклад: Якщо введено "Іван" і "25", то програма повинна вивести: "Привіт, Іван! Через рік тобі буде 26 років".

_Підказка. Щоб отримати те, що введе користувач, використовуй вбудований метод `input()` ([офіційна документація](https://docs.python.org/3/library/functions.html#input))._

_Підказка 2. Не забудь, що вік – це не текстовий тип даних, а число. Щоб зробити його числом, використай метод `int()`._
